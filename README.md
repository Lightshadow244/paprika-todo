# Paprika-ToDo

A simple ToDo app built with Svelte.

## Data storage
All ToDo data is stored in your cookies.

## Features
- Create multiple ToDo lists
- Add ToDo items:
  - `bullet point`
  - `textarea`
  - `checkbox`
- While editing a ToDo, you can:
  - Clear all ToDo items
  - Clear only checked checkbox items
  - Shuffle the ToDo items
